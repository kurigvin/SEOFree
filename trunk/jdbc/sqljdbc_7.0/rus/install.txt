Инструкции по установке
Драйвер Microsoft JDBC 7.0 для SQL Server


ВВЕДЕНИЕ
------------
Здесь представлены инструкции по распаковке файлов sqljdbc_<версия>_enu.exe и 
sqljdbc_<версия>_enu.tar.gz.


WINDOWS
-------
1. Прочтите файл license.txt. Скачивание и использование драйвера Microsoft JDBC 7.0 для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<версия>_enu.exe во временный каталог.

3. Запустите файл sqljdbc_<версия>_enu.exe.

4. Укажите каталог установки, когда будет предложено. Рекомендуется распаковать
   этот ZIP-файл в папку %ProgramFiles%, используя каталог по умолчанию:

	Microsoft JDBC DRIVER 7.0 for SQL Server


UNIX
----
1. Прочтите файл license.txt. Скачивание и использование драйвера Microsoft JDBC 7.0 для SQL Server означает принятие этих условий лицензии.

2. Загрузите файл sqljdbc_<версия>_enu.tar.gz во временный каталог.

3. Чтобы распаковать сжатый TAR-файл, перейдите в каталог, куда нужно
   распаковать драйвер, и введите следующую команду:

	gzip -d sqljdbc_<версия>_enu.tar.gz

4. Чтобы распаковать TAR-файл, переместите его в каталог, где нужно установить драйвер, 
   и введите следующую команду:

	tar -xf sqljdbc_<версия>_enu.tar
